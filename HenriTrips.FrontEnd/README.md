# HenriTrips

Full-stack travel planning platform built with:

- ASP.NET Core 8 Web API
- Angular
- SQL Server
- Docker
- Clean Architecture
- JWT Authentication

---

## Features

- User authentication
- Role-based authorization
- Travel guide management
- Activity management
- User invitations and permissions
- Responsive Angular frontend
- Dockerized deployment

---

## Project Structure

HenriTrips.FrontEnd -> Angular frontend
HenriTrips.FullStackClean -> Clean Architecture backend
database -> SQL scripts
docs -> Documentation
screenshots -> UI previews

---

## Run With Docker

docker compose up --build
